### Backend developer


Java, Kotlin 과 Spring 프레임워크를 기반으로 다양한 플랫폼 서비스를 개발한 백엔드 개발자로, 개발 프로세스의 처음부터 끝까지 참여하여 서비스를 성공적으로 오픈하고 운영한 경험을 보유하고 있습니다.<br/>
